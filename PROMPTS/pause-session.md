# ПАУЗА СЕССИИ

Перед тем как я уйду — зафиксируй текущее состояние работы.

Сделай следующее:

1. **Опиши одной фразой** — что мы делали в этой сессии
2. **Что сделано** — список завершённых шагов (кратко, bullet points)
3. **Где остановились** — последний незавершённый шаг, на каком моменте пауза
4. **Что следующее** — первое действие при возобновлении (одна конкретная задача)
5. **Контекст** — любые важные детали, решения, договорённости которые нужно помнить
6. **Сохрани в файл** — запиши всё это в `PROMPTS/session-state.md`

После сохранения скажи: "Сессия на паузе. Для продолжения используй промпт `resume-session.md`"
