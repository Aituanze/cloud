# agnt.24 App

Мобильное приложение для риэлторов Алматы. Помогает находить объекты собственников за 10 секунд.

## Описание

TikTok-стиль листание объектов + карта районов с пузырями. Доступ к архивным объявлениям Krisha.kz с номерами собственников (через ilvo pro). Система фиксации объектов внутри агентства.

## Функции

- **Карта Pulse Map** — единая AI-сгенерированная сцена «город в миниатюре» (ТВ-башня, горы, ЦУМ, MEGA и др.) на фоне вместо нарисованных гор/улиц; районы — белые бейджи-счётчики поверх (не пузыри, не фото в кружках). Горы сверху (юг вверху).
- **Режимы**: Продажа / Аренда / Архив
- **Архив**: объекты с истёкшей публикацией на Krisha.kz — с номерами собственников
- **TikTok-лента**: scroll-snap карточки, листать вверх
- **Фиксация «В базу»**: резервирует объект за риэлтором. Коллеги видят кто взял в работу.
- **Reveal телефона**: номер собственника скрыт — открывается нажатием
- **Фильтры**: тип недвижимости + ценовой диапазон от/до
