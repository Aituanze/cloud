## Три слоя продукта

### Слой 1 — Рынок (в продакшене ✅)
- Pulse Map: пузырьковая карта районов, счётчик "сейчас"
- Режим Архив: снятые объявления за 7 дней (пунктирные пузыри)
- TikTok-фид с рыночным дифференциалом (% от рынка)
- Фильтры: тип дома, комнатность
- Деплой: https://aituanze.github.io/cloud/

### Слой 2 — Работа (следующий этап)
- CRM-воронка: Новый лид → Контакт → Показ → Оффер → Сделка
- База клиентов с историей контактов
- Планировщик показов с гео-маршрутом
- Клиентская подборка (портал для покупателя)
- Шаблоны договоров по КЗ-праву
- Двойная валюта ₸ / $

### Слой 3 — AI-движок (дальний горизонт)
- AI-матчинг: параметры покупателя → топ объектов
- Telegram-бот: принимает лиды → в CRM автоматически
- Поведенческий скоринг лидов (кто реально купит)
- AI-генерация описаний (русский / казахский)
- Предиктивная аналитика: кто продаст в ближайшие 6 мес
- Conversational search вместо фильтров (2027-горизонт)

## Стратегические окна

### Окно 1 — СРО (май 2026, ~10 месяцев)
Обязательное членство и лицензия для риэлторов. Часть рынка профессионализируется.  
**Действие:** до мая 2026 — первые платящие клиенты в одном агентстве.

### Окно 2 — Распад трафик-монополии порталов (2027)
Agentic AI (Zillow AI Mode, Redfin Sierra) делает агрегаторы менее релевантными.  
Pulse Map как прямой доступ к рынку становится точкой входа.

### Окно 3 — Конкуренты Krisha (анонс 2025, появятся 2027-2028)
При множестве порталов независимый агрегатор выигрывает у закрытых экосистем.

## Данные и технический стек

### Текущий пайплайн (работает)
```
krisha.kz → krisha-parser → listings.db → build_app_data.py → listings.js → agnt-24-app
```

### Следующие поля для парсинга
| Поле | krisha param | Ценность |
|---|---|---|
| `condition` | описание объявления | фильтр "состояние" |
| `deposit` | поле депозита | аренда |
| `who` | `das[who]` | хозяин vs агент |
| `mortgage` | `das[mortgage]` | залоговая |
| `yearBuilt` | год постройки | фильтр по году |

### Стратегия парсинга
- sitemap.xml → список ID → `/a/show/{id}` (оптимально)
- reCAPTCHA при интенсивном обходе → задержки + ротация UA
- Телефон скрыт без авторизации (не парсить)

## MVP для одного агентства — фокус

1. Онбординг агентства (5-20 агентов)
2. Pulse Map как общий экран "пульс рынка"
3. Архив объявлений — звонки собственникам, кнопка "В базу"
4. База (Лента) — мои объекты в работе
5. Уведомления при новых объектах по критериям агента

**Монетизация MVP:** подписка агентства (не per-seat), ~$50-100/мес/агентство.
