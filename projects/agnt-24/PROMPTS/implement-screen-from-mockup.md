# Промпт: Реализовать экран по макету

Реализуй изменения для **Screen [N]** по макету `projects/agnt-24/inbox/screen/[ФАЙЛ].jpg`.

**Процесс:**
1. Read макет — зафиксируй все аннотации и статус ("ОБНОВЛЕНО" / "ПОДТВЕРЖДЕНО")
2. Read текущий код: `index.html` + соответствующая секция `styles.css` + `app.js`
3. Составь конкретный список дельт (что именно отличается)
4. Реализуй каждую дельту минимально — только то что требует макет
5. Не рефакторить соседний код

**Правила дизайна:**
- Цвета: использовать CSS-переменные (--ink, --blue, etc.), не хардкодить hex
- Шрифты: цифры Inter 700, текст Inter 500–600, не использовать Inter 900 для текста
- "ПОДТВЕРЖДЕНО" на макете = текущий код уже правильный, не трогать

**После реализации:** описать что было изменено и почему.
