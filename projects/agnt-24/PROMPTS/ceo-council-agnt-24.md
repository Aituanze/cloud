# Промпт: Стратегический консилиум agnt.24

Запусти CEO Council для анализа стратегического вопроса по agnt.24.

**Контекст проекта (одинаковый для всех экспертов):**
```
agnt.24 — мобильный инструмент для риэлторов Алматы (Казахстан)
Текущий продукт: Pulse Map + TikTok-фид + архивный режим + фильтры
Деплой: https://aituanze.github.io/cloud/ (GitHub Pages)
Данные: krisha.kz парсер, 800+ объявлений, авто-обновление через pipeline.py
Telegram-бот: @Aituanzesbot — входящие лиды в группу, Railway deployment

Рынок:
- Алматы: ~5000 сделок/мес
- Конкуренты: LigaPro (3000 юзеров, нет мобайла), Крыша Агенты (закрытая экосистема)
- Нет аналогов pulse map в КЗ
- С мая 2026 — СРО: обязательная лицензия для риэлторов

Стратегия:
- Сейчас тестируем в одном агентстве Алматы
- Цель: весь рынок КЗ, multi-tenant платформа
- Архитектура: mobile-first, PWA, open-source stack

Пробелы которые закрываем (нет ни у кого в КЗ):
- Нативный мобайл
- AI-матчинг покупатель → объект
- Telegram-бот внутри CRM
- Независимость от одного портала
```
