# realty-quick-match — быстрый поиск недвижимости (Алматы)

## Описание

Мобильное веб-приложение для быстрого поиска недвижимости в стиле TikTok/Threads:
меню по районам Алматы, интерактивная 3D-карта объектов (как korter.kz) и
вертикальная лента объявлений. Бесплатно для покупателей, по подписке для агентств.
Лиды уходят агентству в Telegram, репутация участников строится двусторонними оценками.

## Возможности

- **Меню по районам** — 7 районов Алматы (Бостандыкский, Алмалинский, Медеуский, Ауэзовский, Жетысуйский, Алатауский, Талгарский) с акцентом на Алатауский; по 5 категорий (квартиры, дома, участки, коммерческая, дачи) и счётчиками новинок за 24 часа на каждой иконке.
- **3D-карта MapLibre** — интерактивная карта с наклоном и маркерами-ценниками; попап с кратким описанием; клик по маркеру открывает ленту объектов района/категории.
- **Вертикальная лента (TikTok/Threads)** — полноэкранные карточки объявлений со свайпом вверх/вниз; бейдж «новое 24ч» по `first_seen` или localStorage; лайк, сохранение в избранное, шеринг и кнопка заявки на объекте.
- **Лиды в Telegram** — форма заявки с телефоном покупателя; текст лида (цена, район, ссылка) отправляется агентству через deep-link в Telegram.
- **Двусторонний рейтинг и геймификация** — баллы за активность, бейджи для агентов и покупателей, лидерборд; оценка агента (1–5) с отзывом в профиле.
- **Вирусный шеринг** — генерация подписи для соцсетей с ценой, районом и брендом агентства; вотермарка на карточке для распространения.
- **Ипотечный калькулятор (КЗ)** — расчёт ежемесячного платежа по программам 7-20-25, Отбасы и стандартной ипотеке; первый взнос, сумма кредита и срок.
- **Поиск с фильтрами** — район, категория, диапазон цены (**цена от** и **цена до**); результат открывается в ленте.
- **Дедупликация** — в каталог попадает только первое вхождение объекта по URL.
- **Премиальный дизайн** — современный «взрослый» вид: шрифт Plus Jakarta Sans, палитра «глубокий navy + champagne gold», мягкие тени и скругления, чистые линейные SVG-иконки вместо эмодзи, аватары-монограммы (инициалы), стеклянные кнопки в ленте и плавающий таб-бар.

## Запуск

Открыть `index.html` в браузере или поднять локальный сервер:
`python -m http.server 8000` и зайти на http://localhost:8000

## Генерация данных

`data/listings.js` можно сгенерировать из базы парсера:

```
python build_data.py            # берёт ../krisha-parser/listings.db
python build_data.py путь.db    # явный путь к базе
```

Скрипт дедуплицирует объекты по URL, исправляет «Алматауский» → «Алатауский»
и проставляет координаты центра района, если точных нет.
