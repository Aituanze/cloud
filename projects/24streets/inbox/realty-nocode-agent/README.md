# Realty No-Code Agent — ИИ-агент по поиску недвижимости (прототип)

## Описание

No-code/low-code прототип ИИ-агента по поиску недвижимости в Казахстане. За 4
недели собирается рабочий MVP на готовых сервисах, **без программирования**:
объявления с krisha.kz автоматически собираются в таблицу, отображаются в удобном
веб-приложении с фильтрами и избранным, а подходящие варианты приходят
уведомлениями в Telegram.

Это **направление масштабирования**: после проверки спроса ботом-лидогенератором
(`realty-lead-bot`) здесь подбор объектов автоматизируется, чтобы риэлторы
работали быстрее.

## Возможности (MVP)

- Автоматический сбор объявлений с krisha.kz (Title, Price, Address, Rooms, Area,
  Floor, URL, Photo) в Google Таблицу.
- Веб-приложение для риэлторов: карточки объявлений, поиск, фильтры по цене и
  комнатам, избранное.
- Базовый AI-анализ: текстовые теги типа ремонта (формула `REGEXMATCH`),
  опционально — анализ фото через Google Vision.
- Уведомления в Telegram о новых подходящих объектах (по заданным критериям).

## Технологии (no-code стек)

- **Парсинг:** Octoparse (или ParseHub)
- **Хранилище:** Google Sheets
- **Фронтенд:** Glide
- **Автоматизация:** Zapier (или Make)
- **Уведомления:** Telegram-бот (@BotFather)
- **AI:** формулы Google Sheets, опционально Google Vision API

## Этапы (4 недели)

1. **Неделя 1** — настройка парсинга krisha.kz в Google Sheets.
2. **Неделя 2** — веб-приложение в Glide (карточки, фильтры, избранное).
3. **Неделя 3** — AI-метки (тип ремонта, опц. анализ фото).
4. **Неделя 4** — уведомления в Telegram, тест с реальными риэлторами.

Подробные шаги, частые проблемы и ссылки — в [`source-notes.md`](./source-notes.md).

## Дорожная карта (после MVP)

- Заменить Octoparse на собственный парсер на Python (Playwright, ротация прокси) —
  заготовка уже есть как отдельный проект [`../krisha-parser`](../krisha-parser/README.md).
- Перенести данные из Google Sheets в PostgreSQL.
- Регистрация пользователей, разграничение доступа, личные избранные списки.
- Полноценный AI-анализ фото (модель на казахстанских квартирах).
- Интеграция с CRM (AmoCRM, YClients).
