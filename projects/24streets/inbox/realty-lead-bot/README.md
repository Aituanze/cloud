# Realty Lead Bot — Telegram-бот для сбора лидов по недвижимости

## Описание

Telegram-бот для лидогенерации на рынке недвижимости (Алматы, Казахстан). Клиент
описывает боту, какую недвижимость ищет (район, число комнат, бюджет, телефон), а
бот сохраняет заявку в базу и мгновенно уведомляет группу риэлторов. Подбор
объектов выполняется **вручную** риэлторами — без парсинга сайтов, что полностью
легально.

Это **первый этап** бизнеса: быстро запустить поток заявок и проверить спрос,
не вкладываясь в сложную инфраструктуру.

> Стек: Python 3.10+, `python-telegram-bot` v21 (асинхронный API), SQLite.
> Google Sheets — опционально.

## Возможности

- `/start` — приветствие и кнопка «Отправить мой номер».
- Клиент пишет запрос → лид сохраняется в `leads.db` (SQLite).
- В группу риэлторов приходит уведомление с кнопкой **«Беру в работу»**.
- Телефон автоматически извлекается из текста или из контакта.
- Опционально — дублирование лидов в Google Таблицу.

## Установка

1. Установите **Python 3.10+** с [python.org](https://www.python.org/downloads/).
   При установке на Windows отметьте галочку **«Add Python to PATH»**.
2. Откройте терминал в папке `projects/realty-lead-bot` и создайте виртуальное
   окружение:

```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте бота: напишите [@BotFather](https://t.me/botfather) → `/newbot` → получите **токен**.
2. Скопируйте `.env.example` в `.env` и впишите токен:

```
TELEGRAM_TOKEN=123456:ABC-DEF...
```

3. Создайте группу риэлторов, добавьте туда бота. Узнайте ID группы через
   [@getidsbot](https://t.me/getidsbot) и впишите в `.env`:

```
TELEGRAM_GROUP_ID=-1001234567890
```

## Запуск

```bash
python bot.py
```

Бот начнёт принимать сообщения. Лиды сохраняются в `leads.db`.

## Google Sheets (опционально)

Если хотите дублировать лиды в Google Таблицу:

1. В [Google Cloud Console](https://console.cloud.google.com/) создайте проект,
   включите **Google Sheets API** и **Google Drive API**.
2. Создайте **сервисный аккаунт**, скачайте ключ JSON, переименуйте в `credentials.json`
   и положите в эту папку.
3. Создайте Google Таблицу (имя должно совпадать с `GOOGLE_SHEET_NAME`) и предоставьте
   доступ email-адресу сервисного аккаунта (он есть внутри `credentials.json`).
4. В `.env` установите:

```
USE_GOOGLE_SHEETS=true
GOOGLE_SHEET_NAME=Лидогенератор
```

## Безопасность

- Файлы `.env`, `credentials.json`, `leads.db` не публикуйте — они в `.gitignore`.
